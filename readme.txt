Add WooCommerce product options, add-ons, swatches, conditional fields, pricing, and shared inventory to product pages.

== Description ==

OptionBay helps WooCommerce store owners add custom product options and paid add-ons without creating complex variation sets.

Use it for gift wrap, meal builders, swatches, booking choices, custom notes, service add-ons, and product configuration flows. Customers choose options on the product page. OptionBay then carries those choices into the cart, checkout, and order details.

Smart Product Options and Addons gives your store a practical option builder inside the WordPress dashboard. You can create option groups, add fields, set pricing, control visibility, and manage shared add-on inventory from one place.

[Product Page](https://wpanchorbay.com/plugins/smart-product-options-addons/) | [Official Documentation](https://docs.wpanchorbay.com/smart-product-options-addons/) | [Support](https://wpanchorbay.com/support/) | [Live Demo](https://spoa.wpanchorbay.com/?auto-login=true)

= Key Features =

* Create WooCommerce product option groups from Products > Options.
* Add text, textarea, select, checkbox, single checkbox, radio, number, email, color swatch, image swatch, and static content fields.
* Charge flat fees or percentage-based prices for fields and selectable choices.
* Show or hide fields with conditional logic.
* Use shared inventory pools for add-on stock.
* Support option stock reduction by item quantity or by line item.
* Add option weight to WooCommerce cart items for better shipping totals.
* Display selected options in the cart, checkout, and order line items.
* Import and export option groups, inventory data, and plugin settings.
* Start faster with six draft option group templates.
* Use WooCommerce High-Performance Order Storage with declared HPOS compatibility.

== Installation ==

1. Make sure WooCommerce is installed and active.
2. Upload the plugin folder to `/wp-content/plugins/optionbay-product-options-addons-woo/`.
3. You can also install the plugin through Plugins > Add New in WordPress.
4. Activate OptionBay from the Plugins screen.
5. Go to Products > Options to create your first option group.
6. Publish the option group and assign it globally or to a product.
7. Open a product page and test the customer flow.

== Frequently Asked Questions ==

= Does OptionBay require WooCommerce? =

Yes. OptionBay requires WooCommerce. It works with WooCommerce products, cart items, checkout, order line items, and product settings.

= Where do I manage product options? =

Go to Products > Options in your WordPress dashboard. You can also find settings under WooCommerce > Settings > Products > Options.

= Can I use OptionBay instead of WooCommerce variations? =

Yes, for many product option workflows. OptionBay can reduce the need for large variation sets.

Use WooCommerce variations when each choice needs its own SKU, image, stock record, or native variation behavior.

= What field types are included? =

This free version includes text, textarea, select, checkbox, single checkbox, radio, number, email, color swatch, image swatch, and static content fields.

= Can I charge extra for selected options? =

Yes. You can add flat fees or percentage-based prices to fields and selectable choices.

= Does OptionBay support conditional logic? =

Yes. You can show or hide fields based on customer selections.

= Can I manage stock for add-ons? =

Yes. OptionBay includes shared inventory pools for add-on stock. This is useful when the same add-on is offered across more than one product.

= Will selected options appear in the cart and order? =

Yes. OptionBay displays selected options in the cart and checkout. It also saves them to WooCommerce order line items.

= Does OptionBay support WooCommerce HPOS? =

Yes. The plugin declares compatibility with WooCommerce High-Performance Order Storage.

= Are the starter templates published automatically? =

No. Starter templates are created as drafts. You can review, edit, and publish the templates you want to use.
